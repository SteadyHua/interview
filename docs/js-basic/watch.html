Object.defineProperty(user,'name',{
  set：function(key,value){
    // 这也是 Vue 的原理
  }
})

ES6 中

var  user = new Proxy({}, {
  set：function(target,key,value,receiver){

  }
})

可以监听动态增加的属性。例如 user.id = 1
